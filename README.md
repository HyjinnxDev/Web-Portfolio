# Web Portfolio

A workspace for building and managing multiple websites. Each site lives in its own folder under `websites/`.

## Structure

```
websites/
  _template/     Starter files for new projects
  example-site/  Sample site (replace or delete when you start building)
shared/          Reusable assets, styles, and utilities across sites
```

## Adding a new website

1. Copy `websites/_template` to `websites/your-site-name`
2. Edit `index.html`, `styles.css`, and `script.js` in the new folder
3. Commit and push when ready

## Cursor Mobile

Clone this repo on your phone via Cursor Mobile to work on any site from anywhere:

```
https://github.com/HyjinnxDev/Web-Portfolio
```

## Local preview

Open any site's `index.html` in a browser, or use a simple static server:

```bash
npx serve websites/your-site-name
```
